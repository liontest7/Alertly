1. עיקרון המערכת

המערכת לא מסתמכת על API של Dexscreener.

במקום זאת היא מאזינה ישירות ל:

• transaction logs
• program instructions
• liquidity pool events
• swap events

ממקורות:

• Helius Labs WebSockets
• Solana RPC Streams
• Program logs

כך ניתן לזהות:

יצירת זוג מסחר

הוספת liquidity

swaps גדולים

תשלומי promotion

פעילות whales
